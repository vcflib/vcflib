# NAME

**vcfcommonsamples**

# SYNOPSIS

**vcfcommonsamples** <vcf file> <vcf file>

# DESCRIPTION

Generates each record in the first file, removing samples not present in the second



# OPTIONS

```


Type: transformation

```





# EXIT VALUES

**0**
: Success

**not 0**
: Failure
