# NAME

**vcfremovesamples**

# SYNOPSIS

**vcfremovesamples** <vcf file> [SAMPLE1] [SAMPLE2] ...

# DESCRIPTION

outputs each record in the vcf file, removing samples listed on the command line





# EXIT VALUES

**0**
: Success

**not 0**
: Failure
