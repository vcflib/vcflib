# NAME

**vcfkeepinfo**

# SYNOPSIS

**vcfkeepinfo** <vcf file> [FIELD1] [FIELD2] ...

# DESCRIPTION

To decrease file size remove INFO fields not listed on the command line



# OPTIONS

```


Type: transformation

```





# EXIT VALUES

**0**
: Success

**not 0**
: Failure
