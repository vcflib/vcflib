# NAME

**vcfclassify**

# SYNOPSIS

**vcfclassify** <vcf file>

# DESCRIPTION

Creates a new VCF where each variant is tagged by allele class: snp, ts/tv, indel, mnp



# OPTIONS

```


Type: transformation

      

```





# EXIT VALUES

**0**
: Success

**not 0**
: Failure
