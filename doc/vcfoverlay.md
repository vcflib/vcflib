# NAME

**vcfoverlay**

# SYNOPSIS

**vcfoverlay** [options] [<vcf file> ...]

# DESCRIPTION

Overlay records in the input vcf files with order as precedence.



# OPTIONS

```

options:
    -h, --help       this dialog
    -v, --version    prints version


```
