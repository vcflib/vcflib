# NAME

**vcfcountalleles**

# SYNOPSIS

**vcfcountalleles** <vcf file>

# DESCRIPTION

Count alleles





# EXAMPLES

```

Example:

**vcfcountalleles** samples/scaffold612.vcf

Type: statistics

      

```



# EXIT VALUES

**0**
: Success

**not 0**
: Failure
